Extension Tools

不是看板，是可以操作的 Web 工具箱。

每個入口都連到一個 App 端不適合塞進手機的小型工作台。

01 任務模板產生器

把考試、報告、健康目標拆成可派發的每日任務。

 02 家長共同目標邀請

設定權限、語氣與鼓勵卡，孩子同意後才啟用。

 03 團體挑戰建立器

建立班級、企業、讀書會的步數或專注挑戰。

 04 商城活動配置器

規劃套裝、價格、稀有度與活動期間。

 05 Demo 場景產生器

一鍵生成專題發表用腳本和匿名資料條件。

 06 星球建築規劃器

把下週目標轉成城市建築、事件與解鎖條件。

產品定位

App 是每日操作，Web 是長期價值。

如果 Web 只複製 App 的任務列表和分數，那會很弱。這裡改成提供大螢幕才適合看的資料、 家長與團體角色、活動營運工具，以及發表時能一眼理解系統的展示面。

資料流
  • App 完成任務產生分數、幣、健康與自律房資料
  • Web 整合分析轉成趨勢、風險、週報、星球成長
  • 角色互動家長、老師、團隊、營運者依權限查看
